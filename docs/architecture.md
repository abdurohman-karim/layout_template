# Architecture

Проект намеренно простой и плоский: один HTML-вход, один SCSS-entry, минимум инструментов. Это позволяет быстро стартовать и масштабировать структуру без сборочных монстров.

## Основные директории

- `assets/` — статические ресурсы (шрифты, изображения).
- `styles/` — SCSS-архитектура и файл входа `main.scss`.
- `js/` — JS-скрипты.
- `index.html` — основной HTML.

## SCSS-слои

- `abstracts/` — токены, миксины, брейкпоинты.
- `base/` — reset, базовые стили, шрифты.
- `layout/` — контейнеры и сетки.
- `components/` — независимые UI-блоки.
- `pages/` — стили страниц (если нужно).
- `utilities/` — утилитарные классы.

## Генерация CSS

`styles/main.scss` компилируется в `styles/main.css`. Файл `main.css` считается артефактом сборки и генерируется скриптами npm.
