# Getting Started

## Требования

- Node.js 18+
- npm 9+

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Команда запускает watch-компиляцию SCSS в `styles/main.css`.

## Production сборка

```bash
npm run build
```

## Проверка качества

```bash
npm run lint:styles
```

## Форматирование

```bash
npm run format
```
